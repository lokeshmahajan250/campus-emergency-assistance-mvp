# Campus Emergency Assistance Platform

## Problem
Students often face sudden financial emergencies on campus such as medical needs, academic fees, or personal issues.

## Solution
A simple web-based platform where students can submit emergency financial assistance requests.

## Current Features (Frontend Version)
- User can enter name, problem, and required amount
- Displays submitted request instantly
- Clean and simple user interface

## How it Works
1. User fills the form
2. Clicks submit
3. Request details are displayed on screen

## Tech Stack (Current)
