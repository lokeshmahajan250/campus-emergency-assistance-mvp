# Campus Emergency Assistance Platform (MVP)

This repository contains the documentation for our hackathon project.

## Problem
Students often face sudden financial emergencies on campus such as medical needs, academic fees, or personal issues. Existing solutions are slow, risky, or inaccessible.

## Solution
A secure campus-based digital platform where students can request small emergency financial assistance using already verified college records.

## MVP
The MVP is implemented as a no-code web-based prototype to demonstrate the real workflow.
